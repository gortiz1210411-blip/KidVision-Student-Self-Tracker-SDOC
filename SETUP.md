# Setup

KidVision is configured for district-controlled student data using OneDrive or local review files.

## Recommended Path

1. Install dependencies with `npm install`.
2. Run `npm run dev`.
3. Follow `ONEDRIVE_SETUP.md` for Microsoft 365 integration.
4. Use the local review bundle in `public/KidVision_Local_App_Windows_v13_vocab/` when a static review build is needed.
