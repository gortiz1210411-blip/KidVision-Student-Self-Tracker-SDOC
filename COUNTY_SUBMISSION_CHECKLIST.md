# KidVision County Submission Checklist

Use this checklist when sharing KidVision for district review.

## Branch Readiness

- [ ] The shared branch contains the exact student-facing build demonstrated to district reviewers.
- [ ] Student data remains in district-controlled OneDrive or local district-managed files.
- [ ] Developer-only tools are disabled in the shared branch.
- [ ] No local secrets are tracked in git.

## Send With The Review

- `README.md`
- `ONEDRIVE_SETUP.md`
- `DEPLOYMENT.md`
- `VERIFICATION_CHECKLIST.md`
- Local launch instructions or live review instructions
