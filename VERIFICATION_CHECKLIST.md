# Verification Checklist

## Share-Readiness

- [ ] The demonstrated student-facing build is committed to the shared branch.
- [ ] Developer-only routes are disabled or removed from the shared branch.
- [ ] Local secrets are not tracked in git.
- [ ] District review guidance points to OneDrive/local storage only.

## Validation Commands

```bash
npm run build
```

Optional:

```bash
npm run lint
```
